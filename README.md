## Genre-Based Movie Recommendation System
---

## Project Description
The user enters one or more movie genres (for example: Action, Comedy, Drama).  
Based on this input, the system recommends the **top matching movies** using genre similarity and movie ratings.

This project demonstrates:
- Data preprocessing
- One-hot encoding
- Cosine similarity
- Basic recommendation system logic
